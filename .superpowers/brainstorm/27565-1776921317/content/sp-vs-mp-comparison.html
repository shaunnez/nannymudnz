🔴 Missing in MP

Vitals table — HP, Armor, MR, Move, resource max rows in side panel
VIEW DETAILS button — opens full guild dossier modal
ULT section — ultimate ability name, combo, and description at panel bottom
Guild bio — shown in side panel (SP), not just the center strip
Stat bars — 6-stat pip chart in center strip (MP shows bio text instead)
Active panel highlighting — background tint when panel is "your" cursor

🟡 Size / style mismatches

Tile size — SP 190px · MP 175px (15px smaller)
Tile gap — SP 20px · MP 16px
Tile name font — SP 20px · MP 14px (below the tile)
Tile marker font — SP 20px · MP 10px (P1 / OPP overlays)
Panel role label — SP 14px · MP 11px
Lock button — MP uses raw <button>, SP uses shared AccentBtn
